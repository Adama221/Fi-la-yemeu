# Projet Samabutik (E-Commerce)

## Contexte
Application e-commerce complète avec tableau de bord administrateur, gestion des produits, commandes, et paiements (intégration Wave/Orange Money simulée).

## Backend
- Express.js (serveur complet sur le port 3000)
- SQLite (better-sqlite3)
- MCP intégré (Model Context Protocol)
- Auth JWT

## Frontend
- React 18, Vite, Tailwind CSS
- Design responsive
- Tableau de bord admin (gestion produits, commandes, design, affiliés)

## Raccourcis & Commandes
- Lancement de dev : `npm run dev`
- Lancement de build : `npm run build`
- Vérification MCP : `GET /mcp/messages`
